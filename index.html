Teste de acesso ao SharePoint via Microsoft Graph

Faz login com sua conta Microsoft (popup) e testa a busca da lista "Controle de Clientes" no SharePoint. Roda 100% no seu navegador — o login é direto com a Microsoft, eu não vejo nada disso.
O que fazer com o resultado: depois de rodar o passo 4, tira print (ou copia) do JSON que aparecer — preciso ver os nomes exatos das colunas da lista (Empresa, CNPJ, Situação, Grupo Empresarial etc.) pra montar a busca certa no dashboard final.
